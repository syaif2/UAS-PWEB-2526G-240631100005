## Deskripsi
Aplikasi To Do List merupakan aplikasi berbasis web sederhana yang dibuat menggunakan HTML, CSS, PHP Native, dan MySQL. Aplikasi ini digunakan untuk mengelola daftar tugas dengan fitur tambah, lihat, edit, hapus, dan pencarian data.

## Fitur
- Menampilkan daftar tugas
- Menambah data tugas
- Mengedit data
- Menghapus data
- Mencari tugas berdasarkan judul

## Struktur Database

Database : todo_list

Tabel : tugas

Field:

- id
- judul_tugas
- status
- tanggal

## Cara Menjalankan

1. Install XAMPP.
2. Aktifkan Apache dan MySQL.
3. Import file database.sql ke phpMyAdmin.
4. Simpan project pada folder htdocs.
5. Jalankan melalui:

http://localhost/WEB_UAS/
